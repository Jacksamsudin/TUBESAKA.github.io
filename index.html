Studi Kasus

Studi kasus yang digunakan adalah perhitungan total konsumsi listrik harian berdasarkan data penggunaan listrik (kWh). Permasalahan ini diselesaikan menggunakan algoritma iteratif dan rekursif untuk dibandingkan dari sisi efisiensi.

Input Data Konsumsi Listrik



Deskripsi Algoritma

Algoritma Iteratif

Algoritma iteratif menggunakan perulangan untuk menjumlahkan seluruh data konsumsi.

Kompleksitas Waktu: O(n)

Algoritma Rekursif

Algoritma rekursif menggunakan pemanggilan fungsi berulang hingga kondisi dasar.

Kompleksitas Waktu: O(n)

Grafik Running Time (Realtime)

Grafik akan diperbarui secara langsung setiap kali pengguna menekan tombol Hitung Total.

Analisis Perbandingan

Walaupun kedua algoritma memiliki kompleksitas waktu O(n), algoritma iteratif menunjukkan waktu eksekusi yang lebih stabil dibandingkan algoritma rekursif akibat tidak adanya overhead pemanggilan fungsi.
