Studi Kasus

Studi kasus yang digunakan adalah perhitungan total konsumsi listrik harian berdasarkan data penggunaan listrik (kWh). Permasalahan ini diselesaikan menggunakan dua pendekatan algoritma, yaitu iteratif dan rekursif, untuk dibandingkan dari sisi efisiensi.

Deskripsi Algoritma

Algoritma Iteratif

Algoritma iteratif menggunakan perulangan untuk menjumlahkan seluruh elemen data konsumsi listrik satu per satu.

Kompleksitas Waktu: O(n)

Algoritma Rekursif

Algoritma rekursif menggunakan pemanggilan fungsi secara berulang hingga mencapai kondisi dasar (base case).

Kompleksitas Waktu: O(n)

Grafik Perbandingan Running Time

Grafik berikut menunjukkan perbandingan running time algoritma iteratif dan rekursif pada berbagai ukuran input (1 – 10000).
