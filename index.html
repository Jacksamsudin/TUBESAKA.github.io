Studi Kasus

Studi kasus yang diangkat adalah perhitungan total konsumsi listrik harian berdasarkan data penggunaan listrik (kWh) selama beberapa hari. Permasalahan ini diselesaikan menggunakan dua pendekatan algoritma: iteratif dan rekursif.

Input Data Konsumsi Listrik



Algoritma yang Digunakan

1. Algoritma Iteratif

Algoritma iteratif menjumlahkan data konsumsi listrik menggunakan perulangan. Setiap elemen array diproses satu per satu hingga seluruh data dijumlahkan.

Kompleksitas Waktu: O(n)

2. Algoritma Rekursif

Algoritma rekursif menjumlahkan data dengan memanggil fungsi secara berulang hingga mencapai kondisi dasar (base case).

Kompleksitas Waktu: O(n)

Analisis Perbandingan

Berdasarkan hasil pengujian, kedua algoritma memiliki kompleksitas waktu yang sama, yaitu O(n). Namun, algoritma iteratif lebih efisien dalam penggunaan memori karena tidak memerlukan stack pemanggilan fungsi seperti pada algoritma rekursif.
